## 聖經《和合本》全年讀經表

## input, 總共 1,189 章
```bash 
raw.csv
```

## run by cli
```bash 
$ ./raw2plan.sh      # after day 270, read no. of chapter from 3 to 4.
$ ./csv2json.sh      # convert to json for group_by day
$ ./result2json.sh   # result as json, for programmatic use 
$ ./result2csv.sh    # result as csv, for notion use
```

## output
```bash 
result.csv
```
